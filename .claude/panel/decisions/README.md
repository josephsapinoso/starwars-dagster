# Panel decision logs

One file per debate, named `<YYYY-MM-DD>-<topic>.md`, written by the orchestrator
(main-session Claude) after synthesis — never by a panelist.

Each log records: the shared factual brief, one-line per-role verdicts, the explicit
adjudication (who won each argument and why), the single synthesized plan, and any newly
settled constraints. Panelists then bank their own takeaways into their own memory files
during the BANK pass; this log is the neutral record.
